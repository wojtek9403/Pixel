to test the project :

1.Paste PixelHW1 directory into eclipse workspace
2.Rightclick on Main.class in application package
3.Choose "run as" -> "run configuration"
4.Choose Arguments pane and in VM arguments text field type :

	--module-path {"your absolute path to javafx-sdk-16\lib"} --add-modules 	javafx.controls,javafx.fxml


5.Click apply and run !
